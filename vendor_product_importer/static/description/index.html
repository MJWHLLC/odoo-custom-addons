Vendor Product Importer

Automate Product Imports from Multiple Vendors

Features

Multi-Platform Support
Import products from Amazon, eBay, Shopify, and any website using HTML scraping.

Intelligent Pricing
Automatically calculate profitable prices using tiered pricing rules based on cost ranges.

Smart Product Matching
Match products by SKU, barcode, vendor ID, or name similarity to avoid duplicates.

Tiered Pricing Engine

Cost-Based Pricing Tiers
Define multiple pricing tiers based on cost ranges with different markup percentages.

Flexible Pricing Methods
Choose from percentage markup, fixed amount, or custom Python formulas.

Profit Protection
Set minimum profit margins to ensure profitability on all products.

Automation & Scheduling

Weekly Automated Imports
Schedule automatic product imports to run weekly, keeping your catalog up-to-date.

Manual Import Wizard
Import products on-demand with preview and test mode options.

Comprehensive Logging
Track all imports with detailed logs showing success, failures, and errors.

Key Benefits

Save Time

Automate product imports and eliminate manual data entry.

Increase Profits

Ensure profitable pricing with automated tiered pricing rules.

Stay Updated

Keep product information and prices synchronized with vendors.

Getting Started

  1. Install the module and its dependencies (beautifulsoup4, requests, lxml)
  2. Configure your price tiers (4 default tiers are included)
  3. Add a vendor with API credentials or scraping configuration
  4. Test the connection and run your first import
  5. Enable weekly automation for hands-free operation
